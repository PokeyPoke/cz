# CzLearn — Czech Made Conversational 🇨🇿

A mobile-first, offline-ready Progressive Web App for learning Czech through real conversations. Designed for false beginners to reach conversational ability as quickly as possible.

**Live:** [pokeypoke.github.io/cz](https://pokeypoke.github.io/cz/)

## Features

- 🎧 **8 dialogue-based modules** — from Greetings to Directions
- 🗣️ **Shadowing exercises** — listen & repeat authentic Czech phrases
- ✏️ **Fill-in-the-blank dialogues** — interactive practice
- 🃏 **Spaced repetition flashcards** — SM-2 algorithm
- 🔥 **Streak tracking** — stay motivated
- 🏆 **Achievement badges** — unlock as you progress
- 📱 **Mobile-first design** — installable PWA
- 🌙 **Dark mode** — easy on the eyes
- 📡 **Works offline** — cached audio and content

## Tech Stack

- React 19 + TypeScript
- Vite 8
- Tailwind CSS v4
- React Router v6
- vite-plugin-pwa (service worker)
- lucide-react (icons)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

```bash
npm run deploy
```

Or push to `main` — GitHub Actions deploys to GitHub Pages automatically.
